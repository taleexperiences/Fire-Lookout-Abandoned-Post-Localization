# Game Localization

Welcome! 👋

This repository contains the localization files for Fire Lookout Abandoned Post.

Most translations were initially generated using AI and may contain mistakes, awkward phrasing, or inconsistencies. Contributions from native speakers and experienced translators are greatly appreciated.

## How to Contribute

If you'd like to improve a translation:

1. Fork this repository.
2. Edit the appropriate language file(s).
3. Commit your changes.
4. Open a Pull Request.

Please keep your changes focused on translation quality.

## Translation Guidelines

* Do **not** rename or remove translation keys.
* Preserve placeholders such as `{XKey}`, `<Sam>Sam: </>`, `<Speech></>`, or similar variables.
* Keep formatting, punctuation, and line breaks where appropriate.

## Reporting Issues

If you don't know how to use Git, you can still help!

Open an Issue describing:

* The language
* The incorrect text
* Your suggested improvement

## Thank You

Every contribution helps make Fire Lookout Abandoned Post better for players around the world. Thank you for taking the time to improve the translations!
